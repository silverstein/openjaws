# Beach Panic: Jaws Royale - Gameplay Features

## 1. Asymmetric Gameplay
- **Swimmers (up to 11 players)**: Complete viral challenges while surviving
- **The Shark (1 player)**: Hunt swimmers while dealing with modern annoyances

## 2. Objective System - "Content Creator Chaos"
Swimmers must complete absurd objectives:
- Take a selfie with the shark in frame
- Build perfect sandcastle during attack
- Perform TikTok dances on platforms
- Apply sunscreen mid-chase (2 sec invulnerability)

## 3. Character Archetypes
1. **The Influencer** - "Going Live" creates viewer shields
2. **The Boomer Dad** - "Dad Reflexes" can throw other players
3. **The Surfer Bro** - Can surf on shark's wake
4. **The Lifeguard** - Slow-mo Baywatch run
5. **The Marine Biologist** - Bores shark with facts
6. **The Spring Breaker** - YOLO mode chaos

## 4. Dynamic Events
- "Influencer Invasion" - 5 AI influencers join
- "Shark Week Documentary Crew" - Helicopter with narration
- "Viral Challenge Hour" - Everyone must do challenges
